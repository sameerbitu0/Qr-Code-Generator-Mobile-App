
# QR Code Generator Mobile App

# Getting Started

# 1
Run 
1. npm i
2. npx react-native run-android
3. ./gradelw clean

# 2
Run 
1. npm i
2. cd android
3. ./gradelw clean
4. Back to root
5. npx react-native start --reset-cache
6. npx react-native run-android

# Getting Started IOS

# 1
Run 
1. npm i
2. cd ios
3. pod install
